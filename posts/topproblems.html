Solving the "Big List"

The University of Toronto's MAT237 course has a problem set compiled by Ivan Khatchatourian here. As a matter of ego, I intend to write a "solutions manual" to it here.

All constructive criticism is welcomed in the comments section.

Chapter 1: Topologies

$1.$ Let's remind ourselves of the definition of $\mathbb{R}_{\mathrm{usual}}$. This is the set of reals with the following topology: \[U \in \tau \iff \forall x \in U,~\exists \varepsilon>0 :~B_{\varepsilon}(x) \in U.\] We will show that $(a,b)$ satisfies the property on RHS: let $x \in (a,b)$. Then $a< x< b$, so $x=a+ \varphi_1$ and $x=b- \varphi_2$, for some $\varphi_1, \varphi_2 >0$. Let $\varepsilon = \min (\varphi_1, \varphi_2)$. Then $x \in B_{\varepsilon}(x) \subseteq (a,b)$, so $(a,b) \in \tau$.

For $[a,b)$, we can look at $B = B_a(\varepsilon)$ for any $\varepsilon > 0$. Consider $a-\frac{\varepsilon}{2} \in B$, which is less than $a$ and hence not in $[a,b)$. Therefore there is no such $B \subseteq [a,b)$, so $[a,b) \notin \tau$.



$2.$ Let $G$ be the set generated by $\mathcal{B}$ under the operation $\cup$. Then we will show that \[U \in G \iff U \in \mathcal{P}(X).\] We accomplish both directions at once: \[U \in G \iff (\forall x \in U,~\{x\} \in \mathcal{B}) \iff (\forall x \in U,~x \in X) \iff U \subseteq X \iff U \in \mathcal{P}(X).\] Therefore $G=\mathcal{P}(X)$ by definition. For some topology $\tau$, if $\mathcal{B} \in \tau$, $\mathcal{P}(X) \in \tau$. Since open sets are required to be subsets of $X$ we have $\mathcal{P}(X)= \tau$.



$3(a).$ First, let's define the co-finite topology. The co-finite topology $\tau$ on $X$ is \[\tau = \{U \subseteq X:~U = \emptyset ~ \lor ~ |X \setminus U| \in \mathbb{N} \}.\] The co-countable topology is defined similarly.

Let's do the co-finite case and hope our reasoning works for both. Firstly, $\emptyset \in \tau$ by definition, and $X \in \tau$ since $X=\emptyset^c$ and $|\emptyset|$ is finite.

Finite intersections of open sets are open: suppose $U^c,V^c$ are finite. Then $(U \cap V)^c = U^c \cup V^c$ is a union of finite sets and is thus finite; it results that $U \cap V$ is co-finite (open).

Arbitrary unions of open sets are open: suppose $U^c, V^c$ are finite. Then $(U \cup V)^c = U^c \cap V^c$ is an intersection of finite sets and is thus finite; it results that $U \cup V$ is co-finite (open).

We can see that the above reasoning is the same for the co-countable topology.

$3(b).$ We want to show $U \in \tau_{\mathrm{co-finite}} \implies U \in \tau_{\mathrm{co-countable}}$. Suppose $U^c$ is finite of size $n$. Then $U^c$ is countable by definition and we are done.

$3(c).$ We claim that this holds exactly when $|X|$ is infinite. We need only show the following: for infinite $X$, there is a countably infinite subset $U$. If $X$ is countably infinite, $U =X$. Otherwise, we can pick points $a_n \in X$ and get a set of size $|\mathbb{N}|$ by the induction axiom (we needed to use some version of AoC).

$3(d).$ We claim that this holds exactly when $|X|$ is countable. If $X$ is countable, then any subset $Y$ satisfies $|Y| \leq |X| \leq \mathbb{N}$, so by transitivity $Y$ is countable.

Now we need only show the following: for uncountable $X$, there is a subset $U$ with uncountable complement. This is clear since $\{x\}$ would have uncountable complement $X \setminus x$ (where $x$ is an arbitrary element of $X$).

$4.$ The proof boils down to the fact that a countable union of countable sets is countable. We will prove this fact here since it is used often. Consider countable sets $X_1,X_2,...$ and injections into them from $\mathbb{N}$, $f_1,f_2,...$. Let $f_{i_j} = f_i(j)$, so now we only need an injection $\mathbb{N} \to \mathbb{N}^2$, which we can see on a lattice grid by spiralling outward from the origin.

Our counterexample is motivated by the fact that arbitrary unions of countable sets may be uncountable. The simplest example I can give is \[\bigcup_{x \in \mathbb{R}} \{x\} = \mathbb{R} \] \[\implies \bigcap_{x \in \mathbb{R}} \mathbb{R} \setminus \{x\} = \emptyset, \] where the LHS is an intersection of co-countable sets, and the RHS is obviously not co-countable.

$5.$ By definition.

$6.$ The intersection of $\bigcap (a_i, \infty)$ is $(\max(a_i), \infty)$, if it exists. For the union, we want $\bigcup (a_i, \infty)$ is $(\min(a_i), \infty)$, if it exists. These clearly both exist for finite unions/intersections.

Infinite unions: if $a_i$ is bounded below, we're done - just take the infimum $B$ (which is guaranteed to exist) and the union is $(B, \infty)$. If $a_i$ is unbounded below, then the union should be $\mathbb{R}$. This holds because if we consider the sequence of minima $\min(a_1), \min(a_1,a_2), \min(a_1,a_2,a_3)$ then this is also unbounded and diverges to $-\infty$, so the union indeed diverges to $(-\infty, \infty) = \mathbb{R}$.

$7.$ Consider $U=\bigcup_{x \in A} U_x$. If $u \in U$, then it is an element of some $U_x$, which are all subsets of $A$. If $a \in A$, then there is an open set $U_a$ in the union that defines $U$, so $a \in U$. Hence \[x \in A \iff x \in U,\] i.e $U=A$. Therefore $A$ is a union of open sets, and is therefore open.

$8.$ The first statement should only be true for continuous functions.

$9.$ $\tau_1 \cup \tau_2$ is not necessarily a topology on $X$ - consider $X=\{a,b\}$ and topologies $\tau_1 = \{\emptyset, X, \{a\}\}$ and $\tau_1 = \{\emptyset, X, \{b\}\}$, whose union does not contain $\{a,b\}$ so is not a topology.

$\tau = \tau_1 \cap \tau_2$ is a topology on $X$. If $U_i$ are all in $\tau$, then $U_i \in \tau_1$ and $U_i \in \tau_2$. Since $\tau_1,\tau_2$ are topologies we therefore have $\bigcup U_i \in \tau_1$ and $\bigcup U_i \in \tau_2$. Hence $\bigcup U_i \in \tau$.

If $U,V \in \tau$, then $U,V \in \tau_1$ and $U,V \in \tau_2$. Since $\tau_1,\tau_2$ are topologies we have that $U \cap V \in \tau_1$ and $U \cap V \in \tau_2$. Hence $U \cap V \in \tau$.

Clearly $X$ and $\emptyset$ are also in $\tau$, so $\tau$ is a topology.

$10.$ Consider the set $\mathcal{P}(X \setminus U) + X$ for some $U \subseteq X$. This is a topology on $X$ and is distinct every time we can pick a different $U$, which we can actually do uncountably many times. Hence there are uncountably many topologies on any infinite set $X$.

$14(a).$ It would suffice that there is a rational in $B_\varepsilon(x)$ for any $x$ and $\varepsilon>0$. This intuitively is due to the fact that we can find $n \in \mathbb{N}$ with $\frac{1}{n} < \varepsilon$, i.e the difference between rationals can be made arbitrarily small. Consider \[M=\min \left(\left\{m \in \mathbb{N}~:~ \frac{m}{n} > x-\varepsilon\right\}\right),\] which exists by well-ordering. Suppose for contradiction that $\frac{M+1}{n} \geq x+\varepsilon$. Then $\frac{M}{n} \geq x+(\varepsilon - \frac{1}{n}) > x$, but that would mean $\frac{M-1}{n}>x-\varepsilon$, contradicting the definition of $M$. Therefore $x-\varepsilon < \frac{M+1}{n} < x+\varepsilon$, so there is a rational in any open set in $\mathbb{R}_{\mathrm{usual}}$.

$14(b).$ This would imply the existence of uncountably many distinct rationals. However, there is an injection $\mathbb{N} \to \mathbb{N}^2$ given by $0 \mapsto (0,0), 1 \mapsto (1,0), 2 \mapsto (1,1), 3 \mapsto (0,1),$ etc. (one can imagine spiralling out from the origin on a plane of lattice points). Since $|\mathbb{N}^2| \geq |\mathbb{Q}|$, we have that $\mathbb{Q}$ is countable, a contradiction.
