Toy Quantum Supremacy

Here I introduce quantum computing and demonstrate some toy examples.

Conceptually

Inside a computer, logic is done using binary "gates" - for example, a "NOT" gate might take an input and return the opposite. In this simple case, the output is known whenever the input is known. Consider instead a random gate - it takes any input and returns $1$ or $0$ with equal probability. We might represent this output as \[\frac{1}{2}\binom{1}{1},\] showing the proportion of the time the output will be $0$ or $1$. The utility of gates like this is not immediately obvious - it is not even clear whether we could compute with them in a timely manner.

In fact, there are ways of physically implementing random gates like these, by measuring (for example) the spin of a nucleus in an extremely well-controlled environment. Also, this randomness can facilitate algorithms which are provably faster than classical ones.
