The Jacobian Matrix

Our first generalisation of the derivative is for functions $f: \mathbb{R}^m \to \mathbb{R}^n$. (This actually works for normed vector spaces in general.) We first define the "Frechet" derivative, and then show that it acts component-wise and is the same as the "Jacobian matrix".

This is computationally useful, but also helps us see concretely what happens algebraically when we take derivatives.

A definition of the Jacobian matrix is given here, but you might prefer to just watch it fall out.

The Frechet Derivative

The Frechet Derivative

The Frechet derivative is $Df(\mathbf{x})$, where \[\lim_{|\mathbf{h}| \to 0} \frac{\left| f(\mathbf{x}+\mathbf{h})- f(\mathbf{x}) -(Df(\mathbf{x}))( \mathbf{h}) \right|}{|\mathbf{h}|} = 0.\] $Df(\mathbf{x})$ is intuitively the closest linear transformation to $f$ at $\mathbf{x}$. (The norm symbol doesn't mean the same thing each time it is used here.)

The Frechet Derivative Acts Component-wise

Suppose that $f: \mathbb{R}^m \to \mathbb{R}^n$, and let the basis of $\mathbb{R}^n$ be $\{e_i\}$. First, let's show that the derivative acts component-wise (whenever it exists). Firstly, \[Df(\mathbf{x})(\mathbf{h}) = ((Df(\mathbf{x})(\mathbf{h}))_1 e_1, ..., (Df(\mathbf{x})(\mathbf{h})_n) e_n),\] by the definition of a basis. We just need to show that \[(Df_i(\mathbf{x}))(\mathbf{h})\] exists, and that \[Df_i(\mathbf{x})(\mathbf{h}) = (Df(\mathbf{x})(\mathbf{h}))_i.\] For this, we use the fact that $\{e_i\}$ is a basis: if \[\lim_{|\mathbf{h}| \to 0} \frac{\left| f(\mathbf{x}+\mathbf{h})- f(\mathbf{x}) -(Df(\mathbf{x})) \mathbf{h} \right|}{|\mathbf{h}|} = 0,\] then \[ \lim_{|\mathbf{h}| \to 0} \left| \sum_{i=1}^n \frac{1}{|h|}\left( f_i(\mathbf{x}+\mathbf{h})-f_i(\mathbf{x})-(Df(\mathbf{x}))_i(h) \right) e_i \right|= 0 \] (note that the norm $|\cdot|$ doesn't always mean the same one). The norm of a vector is $0$ only if that vector is $0$. A vector is $0$ only if each component is $0$. Therefore each term of the sum is $0$, i.e \[\lim_{|\mathbf{h}| \to 0} \frac{\left| f_i(\mathbf{x}+\mathbf{h})-f_i(\mathbf{x})-(Df(\mathbf{x}))_i(h) \right|}{|h|} = 0.\] By definition, we can therefore write $Df_i(\mathbf{x}) = Df(\mathbf{x})_i$, or "the $i$th component of the Frechet derivative is the Frechet derivative of the $i$th component".



The Main Result

The Frechet Derivative is (coincides with) the Jacobian Matrix

We can now see algebraically why the Frechet derivative is the Jacobian matrix, as follows. Let \[f(x_1,x_2,...,x_m)=(f_1(x_1,x_2,...,x_m), ..., f_n(x_1,x_2,...,x_m)).\] Taking the Frechet derivative of both sides, \[Df(x_1,x_2,...,x_m) = (Df_1(x_1,x_2,...,x_m), ..., Df_n(x_1,x_2,...,x_m))\] since we can distribute the $D$ component-wise. Now, we can distribute the $D$ component-wise again on the RHS: \[Df(x_1,x_2,...,x_m) = ((Df_1(x_1),..., Df_1(x_m))),(Df_2(x_1),..., Df_2(x_m))),..., (Df_n(x_1),Df_n(x_2),...,Df_n(x_m)))\] and the RHS is exactly the Jacobian matrix since $Df_i(x_j) = \frac{\partial f_i}{\partial x_j}$.

Application in Integration

Dr. Trevor Bazett motivates the method cleanly and quickly here. (We treat the determinant of the Jacobian as how much the integral should change under a substitution, allowing us to make substitutions in multiple integrals.) You can use this to derive an alternate expression for e.g \[I = \iint f(x,y) dxdy\] in terms of polar coordinates $r, \theta$.
